# 🎄 즐거운 크리스마스 (Merry Christmas)

![사이트 샘플 이미지](./sample.png)

![Christmas Tree](https://img.shields.io/badge/Christmas-Tree-green) ![Supabase](https://img.shields.io/badge/Supabase-Realtime-emerald)

안녕하세요! 이 프로젝트는 온라인에서 친구들과 함께 크리스마스 트리를 꾸미고 추억을 남길 수 있는 **롤링페이퍼 웹 서비스**예요. 🎅

👉 **[트리 보러 가기](https://suksa.github.io/christmas/)**

## ✨ 주요 기능

이 서비스에서는 이런 것들을 할 수 있어요!

- **📝 메시지 남기기**: 화면의 빈 공간을 클릭하면 예쁜 메모를 트리에 매달 수 있어요.
- **⚡ 실시간 동기화**: **Supabase**를 사용해서 다른 친구들이 남기는 메모가 실시간으로 뿅! 하고 나타나요.
- **🖱️ 자유로운 배치**: 남긴 메모는 드래그해서 원하는 위치로 옮길 수 있어요. (나만의 자리를 찾아보세요!) 다른 사람이 메모를 이동시켜도 실시간으로 동기화되어 보여요.
- **👥 실시간 접속자 표시**: 화면 상단에서 현재 함께 트리를 꾸미고 있는 친구들의 수를 실시간으로 확인할 수 있어요.
- **🎵 배경 음악**: 왼쪽 상단의 스피커 버튼을 누르면 크리스마스 분위기가 물씬 나는 캐롤이 흘러나와요.
- **📱 모바일 지원**: PC뿐만 아니라 스마트폰에서도 편하게 이용할 수 있어요.

## 🛠️ 사용 기술

이 프로젝트는 다음과 같은 기술로 만들어졌어요.

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend (BaaS)**: [Supabase](https://supabase.com/) (Database & Realtime)
- **Hosting**: GitHub Pages

## 💌 사용 방법

1. 위 링크를 클릭해서 접속해주세요.
2. 배경 음악을 켜고 크리스마스 분위기를 즐겨보세요. 🔊
3. 화면의 빈 곳을 **클릭**하면 메모 작성창이 떠요.
4. 닉네임과 메시지를 적고 **저장**을 누르면 트리에 장식이 추가돼요!
5. 내가 쓴 메모를 드래그해서 예쁜 위치로 옮겨보세요.

모두 행복한 크리스마스 보내세요! 🎄🎁
